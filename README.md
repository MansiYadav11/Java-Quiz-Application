This interactive quiz application presents users with multiple-choice questions about Java fundamentals, tracks their answers, and provides a final score with performance feedback.

✨ Features
5 multiple-choice questions on Java concepts

Real-time feedback on answers (Correct/Wrong)

Score calculation with percentage

Performance evaluation (VERY GOOD/AVERAGE/VERY BAD)

User-friendly console interface

🛠️ Technologies Used
Java

Collections Framework (HashMap)

Scanner for user input
